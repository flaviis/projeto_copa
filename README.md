## Sobre o projeto

Este projeto consiste em um dashboard desenvolvido em Python utilizando Streamlit para apresentar informações sobre jogadores convocados para a Copa do Mundo.

O dashboard permite visualizar estatísticas dos jogadores por meio de gráficos, indicadores e filtros, facilitando a análise dos dados de forma simples e interativa.

## Bases de dados

Foram utilizadas duas bases de dados em formato CSV:

- **jogadores_selecoes_.csv**: contém informações dos jogadores, como nome, seleção, posição, idade, clube, gols, assistências, partidas e atributos técnicos.
- **teams.csv**: contém informações complementares das seleções (seleção, grupo, código FIFA e demais dados), utilizadas na integração dos dados.

## Funcionalidades

- Filtro por seleção;
- Filtro por posição;
- Indicadores gerais (KPIs);
- Distribuição dos jogadores por posição;
- Distribuição dos jogadores por faixa etária;
- Ranking dos maiores artilheiros;
- Ranking dos maiores assistentes;
- Média de gols por posição;
- Desempenho médio da seleção em gráfico radar;
- Tabela com os dados dos jogadores.

## Tecnologias utilizadas

- Python
- Streamlit
- Pandas
- Plotly
